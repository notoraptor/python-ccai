Installation and usage:

```bash
git clone https://github.com/notoraptor/python-ccai

cd python-ccai

# Standard installation
pip install .

# Or editable installation (auto-updated when code is updated)
pip install -e .

python -m pyccai.heightmap --help
```

Example:
```bash
python -m pyccai.heightmap 45.728799618063825 -73.97955593662357 45.39774206758956 -73.43528529565617 -o montreal.png
```
